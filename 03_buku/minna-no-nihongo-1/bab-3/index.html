Bab 3

Menunjukkan Lokasi dan Tempat

Belajar kata tunjuk tempat (koko, soko, asoko) dan menanyakan serta menunjukkan lokasi

60 menit
10 Materi
Tingkat: Pemula

Tata Bahasa

Kata Tunjuk Tempat: ここ・そこ・あそこ・どこ

Dalam bahasa Jepang, kata tunjuk tempat dibedakan berdasarkan jarak dari pembicara dan lawan bicara, mirip dengan kata tunjuk benda.

ここ は きょうしつ です。
Koko wa kyoushitsu desu.
Di sini adalah ruang kelas.
そこ は しょくどう です。
Soko wa shokudou desu.
Di sana adalah kantin.
あそこ は じむしょ です。
Asoko wa jimusho desu.
Di sana (jauh) adalah kantor.
トイレ は どこ ですか。
Toire wa doko desu ka?
Di mana toilet?

Partikel の (no) untuk Menunjukkan Lokasi

Partikel "no" dapat digunakan untuk menunjukkan lokasi suatu benda atau tempat.

つくえ の うえ
tsukue no ue
Di atas meja
いす の した
isu no shita
Di bawah kursi
ほん は つくえ の うえ です。
Hon wa tsukue no ue desu.
Buku ada di atas meja.

Pertanyaan dengan なん (nan) dan だれ (dare)

Kata tanya "nan" (apa) dan "dare" (siapa) dapat digabungkan dengan kata tunjuk tempat untuk menanyakan tentang tempat.

ここ は なん ですか。
Koko wa nan desu ka?
Apa ini tempatnya?
あそこ は だれ の へや ですか。
Asoko wa dare no heya desu ka?
Kamar siapa itu di sana?

も (mo) untuk "Juga"

Partikel "mo" digunakan untuk menyatakan "juga" atau "pula" dalam kalimat.

これ も わたし の ほん です。
Kore mo watashi no hon desu.
Ini juga buku saya.
わたし も がくせい です。
Watashi mo gakusei desu.
Saya juga seorang pelajar.

Latihan

1. Bagaimana mengatakan "Di mana toilet?" dalam bahasa Jepang?

トイレ は なん ですか。
トイレ は どこ ですか。
トイレ は だれ ですか。
トイレ は いつ ですか。

2. Kata tunjuk tempat mana yang digunakan untuk tempat yang dekat dengan lawan bicara?

ここ (koko)
そこ (soko)
あそこ (asoko)
どこ (doko)

3. Bagaimana mengatakan "Buku ada di atas meja" dalam bahasa Jepang?

ほん は つくえ です。
ほん は つくえ の した です。
ほん は つくえ の うえ です。
ほん は つくえ の まえ です。

4. Partikel mana yang digunakan untuk menyatakan "juga"?

は (wa)
の (no)
も (mo)
か (ka)

5. Bagaimana mengatakan "Ini juga buku saya" dalam bahasa Jepang?

これ は わたし の ほん です。
これ も わたし の ほん です。
これ は だれ の ほん です。
これ は ほん です。
