Bab 3

Kata Tunjuk Tempat dan Partikel "de"

Belajar kata tunjuk tempat (koko, soko, asoko) dan penggunaan partikel "de" untuk menunjukkan tempat aktivitas

50 menit
7 Materi
Tingkat: Pemula
Unduh Materi
Latihan

Tata Bahasa

Kata Tunjuk Tempat: ここ・そこ・あそこ

Kata tunjuk tempat digunakan untuk menunjukkan lokasi berdasarkan jarak dari pembicara dan lawan bicara, sama seperti kata tunjuk benda.

ここ は きょうしつ です。
Koko wa kyoushitsu desu.
Di sini adalah ruang kelas (tempat pembicara berada).
そこ は しょくどう です。
Soko wa shokudou desu.
Di sana adalah kantin (tempat lawan bicara berada).
あそこ は としょかん です。
Asoko wa toshokan desu.
Di sana adalah perpustakaan (tempat jauh dari keduanya).

Partikel で (de) untuk Tempat Aktivitas

Partikel "de" digunakan untuk menunjukkan tempat di mana suatu aktivitas dilakukan. Partikel ini diletakkan setelah kata tempat.

きょうしつ で べんきょうします。
Kyoushitsu de benkyou shimasu.
Saya belajar di ruang kelas.
しょくどう で たべます。
Shokudou de tabemasu.
Saya makan di kantin.
としょかん で ほん を よみます。
Toshokan de hon o yomimasu.
Saya membaca buku di perpustakaan.

Pola Kalimat dengan Kata Kerja

Dalam bab ini kita mulai menggunakan kata kerja. Pola kalimatnya adalah: [Subjek] wa [Tempat] de [Kata Kerja]. Partikel "o" digunakan untuk menandai objek langsung.

わたし は きょうしつ で べんきょうします。
Watashi wa kyoushitsu de benkyou shimasu.
Saya belajar di ruang kelas.
がくせい は としょかん で ほん を よみます。
Gakusei wa toshokan de hon o yomimasu.
Pelajar membaca buku di perpustakaan.

Percakapan

Yamada
すみません、しょくどう は どこ ですか。
Sumimasen, shokudou wa doko desu ka?
Permisi, kantin di mana?
Tanaka
しょくどう は あそこ です。
Shokudou wa asoko desu.
Kantin ada di sana.
Yamada
たなかさん は どこ で べんきょうします か。
Tanaka-san wa doko de benkyou shimasu ka?
Tanaka belajar di mana?
Tanaka
わたし は きょうしつ で べんきょうします。
Watashi wa kyoushitsu de benkyou shimasu.
Saya belajar di ruang kelas.
Yamada
としょかん で なに を します か。
Toshokan de nani o shimasu ka?
Apa yang Anda lakukan di perpustakaan?
Tanaka
としょかん で ほん を よみます。
Toshokan de hon o yomimasu.
Saya membaca buku di perpustakaan.

Latihan

1. Bagaimana mengatakan "Saya belajar di ruang kelas" dalam bahasa Jepang?

わたし は きょうしつ です。
わたし は きょうしつ で べんきょうします。
きょうしつ で わたし です。
きょうしつ は べんきょうします。

2. Manakah partikel yang tepat untuk menunjukkan tempat aktivitas?

は (wa)
の (no)
で (de)
を (o)

3. Pilih kalimat yang tepat: Anda ingin mengatakan "Saya membaca buku di perpustakaan"

としょかん は ほん を よみます。
ほん で としょかん を よみます。
としょかん で ほん を よみます。
よみます で としょかん を ほん。

4. Kata mana yang digunakan untuk menanyakan "di mana"?

だれ (dare)
なん (nan)
どこ (doko)
いつ (itsu)
